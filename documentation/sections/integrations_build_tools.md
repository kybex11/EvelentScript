### Build Tools

* [Browserify](http://browserify.org) — compile `.es` files with a transform plugin

* [Grunt](https://gruntjs.com) — compile `.es` files with a contrib plugin

* [Gulp](https://gulpjs.com) — compile `.es` files with a stream plugin

* [Parcel](https://parceljs.org) with [transformer-evelentscript](https://github.com/parcel-bundler/parcel/tree/v2/packages/transformers/evelentscript)

* [Rollup](https://rollupjs.org) with [rollup-plugin-evelentscript](https://github.com/lautis/rollup-plugin-evelentscript)

* [Webpack](https://webpack.js.org) — compile `.es` files with a loader plugin
