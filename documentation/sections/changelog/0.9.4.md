EvelentScript now uses appropriately-named temporary variables, and recycles their references after use. Added `require.extensions` support for **Node.js 0.3**. Loading EvelentScript in the browser now adds just a single `EvelentScript` object to global scope. Fixes for implicit object and block comment edge cases.
