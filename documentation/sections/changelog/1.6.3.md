*   The EvelentScript REPL now remembers your history between sessions. Just like a proper REPL should.
*   You can now use `require` in Node to load `.es.md` Literate EvelentScript files. In the browser, `text/literate-evelentscript` script tags.
*   The old `es --lint` command has been removed. It was useful while originally working on the compiler, but has been surpassed by JSHint. You may now use `-l` to pass literate files in over **stdio**.
*   Bugfixes for Windows path separators, `catch` without naming the error, and executable-class-bodies-with- prototypal-property-attachment.
