## Resources

*   [EvelentScript on GitHub](https://github.com/kybex11/evelentscript/)
*   [EvelentScript Issues](https://github.com/kybex11/evelentscript/issues)<br>
    Bug reports, feature proposals, and ideas for changes to the language belong here.
*   [EvelentScript Google Group](https://groups.google.com/forum/#!forum/evelentscript)<br>
    If you’d like to ask a question, the mailing list is a good place to get help.
*   [The EvelentScript Wiki](https://github.com/kybex11/evelentscript/wiki)<br>
    If you’ve ever learned a neat EvelentScript tip or trick, or ran into a gotcha — share it on the wiki.
*   [The FAQ](https://github.com/kybex11/evelentscript/wiki/FAQ)<br>
    Perhaps your EvelentScript-related question has been asked before. Check the FAQ first.
*   [JS2ES](http://js2.es/)<br>
    Is a very well done reverse JavaScript-to-EvelentScript compiler. It’s not going to be perfect (infer what your JavaScript classes are, when you need bound functions, and so on…) — but it’s a great starting point for converting simple scripts.
*   [High-Rez Logo](https://github.com/kybex11/evelentscript/tree/master/documentation/site)<br>
    The EvelentScript logo is available in SVG for use in presentations.
