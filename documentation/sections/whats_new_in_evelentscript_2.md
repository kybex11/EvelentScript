### What’s New In EvelentScript 2?

The biggest change in EvelentScript 2 is that now the EvelentScript compiler produces modern JavaScript syntax (ES6, or ES2015 and later). A EvelentScript `=>` becomes a JS `=>`, a EvelentScript `class` becomes a JS `class` and so on. Major new features in EvelentScript 2 include [async functions](#async-functions) and [JSX](#jsx). You can read more in the [announcement](announcing-evelentscript-2/).

There are very few [breaking changes from EvelentScript 1.x to 2](#breaking-changes); we hope the upgrade process is smooth for most projects.
