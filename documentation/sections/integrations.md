## Integrations

EvelentScript is part of the vast JavaScript ecosystem, and many libraries help integrate EvelentScript with JavaScript. Major projects, especially projects updated to work with EvelentScript 2, are listed here; more can be found in the [wiki pages](https://github.com/evelent-core/evelentscript/wiki). If there’s a project that you feel should be added to this section, please open an issue or [pull request](https://github.com/evelent-core/evelentscript/wiki/%5BHowTo%5D-Update-the-docs). Projects are listed in alphabetical order by category.
