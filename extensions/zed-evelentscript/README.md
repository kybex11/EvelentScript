# EvelentScript for Zed

Language support for EvelentScript (`.es`, `.lites`, `.es.md`) in [Zed](https://zed.dev).

## Install

Copy or symlink this extension into your Zed extensions directory, or install from the repository path during development.

## Supported extensions

- `.es` — EvelentScript source
- `.lites` — literate EvelentScript
- `.es.md` — literate EvelentScript (Markdown)
